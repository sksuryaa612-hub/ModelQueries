# 📱 Responsive Flexbox Layout

A simple and responsive web page built using HTML and CSS Flexbox. The layout displays three content boxes in a row on larger screens and automatically adjusts to smaller screen sizes for a better user experience.

## 🚀 Features

- Responsive design
- Built with HTML and CSS
- Flexbox layout
- Mobile-friendly
- Clean and simple structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries

## 📱 Responsive Behavior

- Desktop: 3 boxes in one row
- Tablet: 2 boxes in one row
- Mobile: 1 box per row

## 🎯 Purpose

This project demonstrates how to create a responsive layout using Flexbox and Media Queries in CSS.
